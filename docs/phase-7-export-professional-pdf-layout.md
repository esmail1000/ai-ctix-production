# Phase 7 Export Professional PDF Layout

This patch upgrades the PDF export generated by `/api/export?format=pdf`.

## What changed

- Replaces the plain text PDF stream with a structured professional report layout.
- Adds a cover page with report identity, source file, generated timestamp, risk score, findings count, and parser confidence.
- Adds executive summary and severity distribution chart.
- Adds an overview page with risk rationale and priority matrix.
- Adds detailed finding cards with severity badges, score, asset, CVE, attack vector, impact, evidence, remediation, and risk factors.
- Adds prioritized recommendation cards.
- Adds consistent green CTIX visual styling, page headers, borders, cards, and page footers.

## Guardrails

- Frontend buttons and Word export remain unchanged.
- The PDF still uses the same real saved report data.
- No mock data, fake counts, placeholder findings, or simulated intelligence were added.
- No backend analysis, NLP, threat intel, graph, or risk scoring logic was changed.
